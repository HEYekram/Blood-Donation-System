# Blood Donation Management System

A Blood Donation Management System is a web-based application designed to manage blood donor information, blood requests, and donation-related activities. This system helps users find suitable blood donors quickly and helps administrators manage donor and request data efficiently.

## Features

* Donor registration
* Donor login and profile management
* Blood request creation
* Search donors by blood group
* Manage donor information
* Manage blood requests
* Admin dashboard
* User-friendly interface
* Secure data management

## Project Objectives

The main objectives of this project are:

* To make blood donor information easily available
* To help patients find blood donors quickly
* To reduce manual work in blood donation management
* To maintain donor and blood request records digitally
* To improve communication between donors and recipients

## How the System Works

1. A donor registers in the system with personal and blood group information.
2. Users can search for donors based on blood group.
3. Users can submit blood requests when they need blood.
4. Admin can manage donors, users, and blood requests.
5. The system helps connect blood donors with people who need blood.
